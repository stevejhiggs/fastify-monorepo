# Temporal Worker

A Temporal worker that executes workflows and activities. This worker is part of a Temporal app generated by `pnpm generate:temporal-app`.

## Getting started

1. Start a Temporal server: `temporal server start-dev`
2. Install dependencies from the repository root: `pnpm install`
3. Start the worker: `pnpm dev` (from this directory)

## Structure

```
src/
  index.ts       # Worker entry point (connects to Temporal, registers activities)
  workflows.ts   # Re-exports workflows from the shared workflows package
```

## How it works

- **Workflows** are imported from the shared `@repo/<name>-workflows` package and re-exported via `workflows.ts` for the Temporal SDK's Webpack bundler (`workflowsPath`)
- **Activities** are imported from `@repo/<name>-workflows/activities` and registered directly with the worker
- **Logging** uses `@repo/logging` with `@repo/temporal`'s logger adapter so all Temporal SDK logs flow through structured pino logging

## Environment Variables

- `LOG_LEVEL` - Logger level (default: `info`)
- `TEMPORAL_ADDRESS` - Temporal server address (default: `localhost:7233`)
